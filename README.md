# Bootstrap PHP CRUD com Firebird

Este é um projeto de estudo de um CRUD (Create, Read, Update, Delete) utilizando PHP com arquitetura simples, Bootstrap para a interface e banco de dados Firebird.

## 🚀 Tecnologias Utilizadas
* PHP 8.x (PDO)
* Bootstrap 5
* Banco de Dados Firebird 3.0
* FlameRobin (Administração do Banco)

## ⚙️ Configuração do Banco
O projeto utiliza a porta **3050** para conexão com o Firebird.
Caminho do banco configurado: `C:\database\BANCO.FDB`

## 📂 Estrutura do Projeto
* `/config`: Configurações de conexão.
* `/includes`: Cabeçalho e rodapé reutilizáveis.
* `/public`: Scripts de teste e arquivos públicos.
* `code.php`: Lógica de processamento de dados (Controller).
* `index.php`: Página principal com listagem e Modal.
